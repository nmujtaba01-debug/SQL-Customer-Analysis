# SQL Customer Analysis Project

## About
Real SQL business analysis of a retail customer 
database using W3Schools Northwind dataset.

## Business Questions Answered
1. Total number of customers
2. Top 5 countries with most customers
3. European countries customer distribution
4. Cities with more than 2 customers
5. Customers with names starting A to D
6. Customers with more than 3 orders
7. Customers who ordered in 1997
8. Orders placed each year
9. Shipper with most deliveries
10. German customers order count

## SQL Skills Demonstrated
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- JOIN across multiple tables
- Aggregate functions COUNT, AVG, MAX, MIN
- LIKE for pattern matching
- UNION for combining results
- TOP for limiting results
